## عن رَفٌّ

**رَفٌّ** برنامج مكتبي عربي لإدارة وفهرسة المكتبات، صُمم للمكتبات الصغيرة، المدارس، المراكز التعليمية، المساجد، المبادرات الخيرية، والباحثين الذين يريدون نظامًا بسيطًا وسريعًا بدل الجداول المبعثرة والبحث اليدوي.

هذا الإصدار **0.3.0 Beta** مخصص للتجربة المبكرة وجمع الملاحظات قبل الوصول إلى النسخة المستقرة.

---

## أبرز المميزات

| المجال | ما يقدمه رَفٌّ |
|---|---|
| الفهرسة | تسجيل اسم الكتاب، المؤلف، دار النشر، التصنيف، الطبعة، سنة النشر، الحالة، الملاحظات |
| البحث | بحث فوري في العنوان، المؤلف، دار النشر، التصنيف، والرقم المرجعي |
| العربية | تجاهل التشكيل والتطويل وفهم اختلافات الحروف مثل: أ، إ، آ، ا، ى، ي، ة، ه |
| الأرقام المرجعية | إنشاء رقم مرجعي تلقائي لكل كتاب لتسهيل ترتيبه داخل المكتبة |
| الإعارة | تحديد حالة الكتاب: متاح أو معار، مع تسجيل اسم المستعير |
| التقارير | إحصائيات عامة، أكثر التصنيفات، أكثر دور النشر |
| التصدير | تصدير المكتبة إلى JSON وCSV وTXT وPDF |
| الأمان | يعمل محليًا على جهازك ولا يرسل بيانات مكتبتك لأي خادم خارجي |

## التحميل

حمّل آخر إصدار من صفحة الإصدارات الرسمية:

[GitHub Releases](https://github.com/Hzifa33/raff/releases/latest)

بعد التحميل:

```text
1. شغّل ملف التثبيت أو فك ضغط النسخة المحمولة إن وُجدت.
2. افتح برنامج رَفٌّ.
3. أضف أول كتاب أو استورد نسخة احتياطية.
```

## التحقق من سلامة الملف

استخدم قيمة **SHA256** التالية للتحقق من أن الملف الذي حمّلته هو نفس الملف المنشور رسميًا:

```text
222162d2b326516b9f037690cc4e8d67da44f35150e4164493386d6c9b40ae9c
```

للتحقق على Windows، افتح موجه الأوامر داخل مجلد التحميل واكتب:

```bat
certutil -hashfile Raf_Setup_0.3.0.exe SHA256
```

إذا كان اسم الملف مختلفًا، استبدل `Raf_Setup_0.3.0.exe` باسم الملف الموجود لديك.

> ملاحظة: لا تكتب البصمة بهذا الشكل `sha256:hash` داخل README؛ الأفضل وضعها في سطر مستقل أو داخل كتلة كود كما هو موضح أعلاه.

## ملاحظة حول Windows SmartScreen

قد يعرض Windows رسالة مثل **Unknown Publisher** لأن البرنامج جديد وغير موقّع بشهادة Code Signing مدفوعة.

هذا لا يعني بالضرورة أن البرنامج ضار. لتقليل المخاطر:

```text
حمّل البرنامج فقط من GitHub Releases الرسمي
تحقق من SHA256 قبل التشغيل
لا تعتمد على نسخ مرسلة من مصادر مجهولة
```

## الخصوصية ومكان البيانات

رَفٌّ يعمل محليًا على جهازك.

لا يقوم البرنامج بـ:

```text
جمع بيانات المستخدم
رفع بيانات المكتبة إلى الإنترنت
تتبع الاستخدام
عرض الإعلانات
إرسال الكتب أو أسماء المؤلفين إلى خوادم خارجية
```

تُحفظ بيانات المكتبة في ملف محلي داخل مجلد بيانات التطبيق الخاص بالمستخدم، باسم:

```text
raff-library.json
```

## للمطورين

المشروع مبني باستخدام **Electron** و **JavaScript**.

### تشغيل المشروع محليًا

```bash
npm install
npm start
```

### بناء نسخة Windows

```bash
npm run dist:win
```

سيتم إخراج ملفات البناء داخل مجلد:

```text
release
```
