# رَفٌّ

### نظام عربي أنيق لفهرسة وإدارة المكتبات

## ما هو رَفٌّ؟

**رَفٌّ** برنامج مكتبي مجاني وخيري لإدارة وفهرسة المكتبات العربية، صُمم للمكتبات الصغيرة، المدارس، المراكز التعليمية، المساجد، المبادرات الخيرية، والباحثين الذين يريدون نظامًا واضحًا وسهلًا بدل الجداول المبعثرة والبحث اليدوي.

الإصدار الحالي **0.3.0 تجريبي**، وهو مخصص للاختبار المبكر وجمع الملاحظات قبل الإصدار المستقر.

---

## لماذا رَفٌّ؟

| الميزة | الفائدة |
|---|---|
| بحث عربي ذكي | يتجاهل التشكيل والتطويل ويفهم اختلافات مثل: أ، إ، آ، ا، ى، ي، ة، ه |
| فهرسة منظمة | اسم الكتاب، المؤلف، دار النشر، الرقم المرجعي، التصنيف، الحالة، الملاحظات |
| أرقام مرجعية تلقائية | إنشاء رقم منظم لكل كتاب لتسهيل الوصول إليه داخل المكتبة |
| إدارة حالة الكتاب | معرفة الكتب المتاحة والمعارة وتسجيل اسم المستعير |
| واجهة عربية كاملة | تصميم RTL واضح ومريح للاستخدام اليومي |
| تقارير وتصدير | تصدير بيانات المكتبة إلى JSON وCSV وTXT وPDF |
| نسخ احتياطي واستعادة | حفظ نسخة احتياطية من بيانات المكتبة واسترجاعها عند الحاجة |
| يعمل محليًا | لا يحتاج إلى إنترنت ولا يرسل بياناتك إلى أي خادم خارجي |

## التحميل

حمّل آخر إصدار من صفحة الإصدارات:

[GitHub Releases](https://github.com/Hzifa33/raff/releases/latest)

بعد التحميل:

```text
1. فك الضغط عن الملف إذا كان Portable.
2. شغّل Raf.exe.
3. ابدأ بإضافة الكتب أو استيراد نسخة احتياطية.
```

قد تظهر رسالة Windows SmartScreen لأن البرنامج جديد وغير موقّع بشهادة مدفوعة. حمّل البرنامج فقط من الرابط الرسمي، وتحقق من قيمة SHA256 قبل التشغيل.

---

## التحقق من سلامة الملف

قيمة SHA256 لإصدار **0.3.0**:

```text
222162d2b326516b9f037690cc4e8d67da44f35150e4164493386d6c9b40ae9c
```

للتحقق على Windows، افتح موجه الأوامر داخل مجلد التحميل واستخدم:

```bat
certutil -hashfile Raf_Portable_0.3.0.zip SHA256
```

إذا كان اسم الملف مختلفًا، استبدل الاسم في الأمر باسم الملف الذي حمّلته.

## الخصوصية

رَفٌّ يعمل محليًا على جهازك.

لا يقوم البرنامج بـ:

```text
جمع بيانات المستخدم
إرسال بيانات الكتب إلى الإنترنت
تتبع الاستخدام
عرض الإعلانات
الاتصال بخوادم خارجية لإدارة بيانات المكتبة
```

تُحفظ بياناتك داخل ملف محلي باسم:

```text
raff-library.json
```

داخل مجلد بيانات التطبيق الخاص بالمستخدم.

## للمطورين

المشروع مبني باستخدام **Electron** و **JavaScript**، مع واجهة عربية مخصصة وملفات محلية لحفظ بيانات المكتبة.

### تشغيل المشروع

```bash
npm install
npm start
```

### بناء نسخة Windows

```bash
npm run dist:win
```

ينشئ البناء ملف تثبيت داخل مجلد:

```text
release
```

كما يحتوي المشروع على GitHub Actions لبناء نسخة Windows تلقائيًا عند إنشاء Tag يبدأ بـ `v`.
